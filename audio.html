Python Audio library

The Audio library combines current audio libraries for use in AI projects. It provides methods to record audio while there is sound, return audio streams of the tone, record and play audio and more. We are still working on natrual audio recognition and learning. Please download and use in your own robotics projects! Some features use the Google speech recognition engine to return recognized text.

Requirements

  • Sound device
  • Sound file
  • Speech recogition
  • wave
  • wavio
  • numpy

Using pip you can install all the dependency libraries. Built using Python 3 Uses wav files only.
Using The Library
You can include the Audio library using the following line:
from soundLib import audio

The library has several functions the user can use.


class = audio(system pathway) -- This declares the audio interface with the pathway to where to save and fid files

recording = class.recordAudio(int_time_of_recording) -- record audio for set time

class.save(filename,recording) -- save the recording to the file specified

class.play(filename) -- play sound file specified

stringText = class.getText(filenameOfAudioWav) -- return the text of spoken wors in a sound file

sample = class.getSample(intSeconds) -- record a sample of audio levels

class.displayLevels(sample) -- print the size of the audio stream items

intThreshold = class.getThreshold() -- return the of the audio

sample = class.getSample(intSeconds) -- Get an array of sound levels for seconds specified

class.recordWhileActive(filename) --record a wave file specified until user stops talking


from soundLib import *

myMic = audio("pathToFile/") #Create a mircophone object
myMic.recordWhileActive("test.wav")
